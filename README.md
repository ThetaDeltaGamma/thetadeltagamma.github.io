# Theta Delta Gamma

**Itinatag:** Abril 24, 2020  
**Mga Nagtatag:** Lawrence Esplana at John Jovi Cordoves

Ang Theta Delta Gamma ay para sa magandang layunin ng pagkakapatiran  
pagkakaroon ng respeto disiplina at paglilingkod sa kapwa.  

Sabay na pagtibayin ang pagkakaisa pagmamalasakit sa bawat kapatid  
pagpapakita ng mabuting pamumuno at pagpapalaganap ng positibong impluwensya  
sa komunidad.  

Layunin din ang pagpapatibay ng kapatiran at nagkakaisa ng samahan.
